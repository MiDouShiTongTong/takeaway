# takeaway
### 项目开发准备
- 项目描述
- 主要的功能
- 主要的技术
- spa 应用
- 采用模块化(ES6 babel)/工程化(webpack, eslint, scss)开发
- API 接口

### 目录结构
- src
  + assets `通用资源文件[css, js, img]`
  + component `非路由的组件`
  + pages `路由的组件`
  + router `路由的模块`
  + store `vuex的模块`
  + api `与后台交互的模块`
  + filters `过滤器的模块`
  + mock `模拟数据接口的模块`
  + App.vue `根组件`
  + main.js `入口文件`
